## 📖 About

**Wallpaper Gallery App** is a modern and responsive React application that displays beautiful wallpapers using the **Picsum Photos API**. It is built to practice React fundamentals, API integration, reusable components, and responsive UI design.

🌐 **Live Demo:**  
https://gallery-projectbymax.netlify.app/

📂 **GitHub Repository:**  
https://github.com/madebymaxx/wallpaper-app-by-maxx

---

## ✨ Features

- 🔍 Browse beautiful random wallpapers
- 🖼️ Images powered by Picsum Photos API
- 📱 Fully Responsive Design
- ⚡ Fast Loading
- 🎨 Clean & Modern UI
- ♻️ Reusable React Components
- 🚀 Built with React + Vite
- 🌍 Dynamic Image Loading

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/madebymaxx/wallpaper-app-by-maxx.git
```

### Navigate to Project

```bash
cd wallpaper-app-by-maxx
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

Open your browser:

```
http://localhost:5173
```

---

## 🌐 API Used

**Picsum Photos API**

```
https://picsum.photos/v2/list?page=1&limit=10
```

✅ No API key required.

## ⚙️ How It Works

1. User opens the application.
2. React renders the UI.
3. The app requests images from the Picsum API.
4. Images are fetched dynamically.
5. Wallpapers are displayed in a responsive gallery.
